Question-1: What are the different ways to select an element in the DOM?
DOM থেকে বিভিন্ন উপায়ে element সিলেক্ট করা যায় নিচে আমার জানা কয়েকটা সিলেক্টর এর নাম দিলাম কারণ আমি এখনো সব গুলো সম্পর্কে জানি না
  • 1. getElementById() tag এর id দিয়ে টার্গেট করে
  • 2. getElementsByClassName() ক্লাস নাম একাধিক থাকতে পারে তাই সবাইকে একসাথে টার্গেট করা যায়।
  • 3. getElementsByTagName() direct tag name দিয়ে সিলেক্ট করা যায়।
  • 4. getElementsByName() direct নাম দিয়ে element ধরা যায়
  • 5. querySelector() বেশি ফ্লেক্সিবল ভাবে কাজ করা যায় এবং CSS সিলেক্টর ব্যবহারকরা যায়।
  • 6. querySelectorAll() বেশি ফ্লেক্সিবল ভাবে কাজ করা যায় এবং CSS সিলেক্টর ব্যবহারকরা যায়।

Question-3: What is event delegation in the DOM?
ইভেন্ট ডেলিগেটে হল একটি মেথর্ড যা চাইল্ড এলিমেন্টের পরিবর্তে প্যারেন্ট এলিমেন্টে ইভেন্ট লিসেনার সেট করে চাইল্ড এলিমেন্টের ইভেন্টগুলো হ্যান্ডেল করে। ইভেন্ট ডেলিগেটে ইভেন্ট বাবলের মাদ্ধমে চাইল্ড এলিমেন্টে ইভেন্ট চালালে প্যারেন্ট প্রযন্ত চোলে যায়, তাই আলাদা আলাদা ইভেন্ট লিসেনার দিতে লাগে না এতে প্রোগ্রাম হালকা হয় এবং দ্রুত রান কর।
Question-4: What is event bubbling in the DOM?
event bubbling হল একটি মেথর্ড যা চাইল্ড এলিমেন্টের মাধ্যমে parent element হয়ে ক্রমশ উপরের দিকে propagate হয়। এটি সাধারণত প্রোগ্রামের ভিতর থেকে বাহিরের দিকে যায় তাই পারেন event elements একই event ক্যাপচার করতে পারে তাই event bubbling চলতেই থাকে তাই এটি থামানোর জন্য আলাদা করে কমান্ড দিতে হয় যাতে কাঙ্ক্ষিত জায়গা প্রজন্ত গিয়ে কাজ কর। event.stopPropagation() দিয়ে নিদিষ্ট জায়গায় কাজ করতে পারি।
Question-5: How do you create, add, and remove elements using JavaScript?

Create Elements

document.createElement() দিয়ে এলিমেন্ট নির্ধারণ করবো
.innerText = " " এ বলে দিতে হবে এলিমেন্টের মধ্যে কি লেখা থাকবে
.appendChild() দিয়ে কমান্ড করে এলিমেন্ট পুশ করে দিব

Add Elements

কিভাবে add করা যায় তা উপরেই বলে দিয়েছি তবে ইটা ছাড়া আরো কিছু কমান্ড আছে যা দিয়ে তৈরী করে element কে add করা যায় যেমন :
.appendChild()
.append()

Remove Elements

.remove() টার্গেটেড element কে সরাসরি এলিমেন্ট মুছে ফেলা যায়
.removeChild() Parent element থেকে নির্দিষ্ট Child element মুছে ফেলা যায়
